# 🚗 Online Customer Support System  

> **Luxury Motor Solutions – Software Process Modeling Project**  
> *BSc (Hons) in Information Technology | IT1060 – Semester II, 2022*

---

## 📌 Project Overview
This repository contains the **Assignment 1** submission for *Software Process Modeling*.  
The project models an **Online Customer Support System** tailored for **Luxury Motor Solutions**, offering premium support services for luxury car owners.  

Key features include:  
- 📝 Account creation, login, and profile management  
- 💬 Chat assistant for troubleshooting and FAQs  
- 📞 Call assistant for real-time support  
- 📊 Feedback collection and management  
- 💳 Payment handling for services and parts  
- 📦 Vehicle parts pricing and stock management  
- 🛠️ Admin and marketing manager functionalities

## 🏗️ System Use Cases
The system is modeled with **10 core use cases**:

1. 📝 Create User Account  
2. 🔍 Search for Service Information  
3. 🔑 Login  
4. ⚙️ Update User Profile  
5. 🗣️ Manage Feedbacks  
6. 👥 Manage User Accounts  
7. 💳 Payment for Service  
8. 🛒 Manage Prices of Vehicle Parts  
9. 💬 Provide Relevant Information (Chat Assistant)  
10. 📩 Forward Customer Feedback to Manager
